To play this game you will need to extract the files using a program such as WinRar or 7Zip.
Once you've installed a program to extract the files and have extracted them, go into the folder and double click FireTrainingSim.exe.

Controls:
Click the left mouse button to control where the player moves to.
Click on a glowing object to pick it up or activate a minigame.
Clicking on a button in the menu lets you control the menu.
